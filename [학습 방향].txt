3. 학습 우선순위

파이썬 기초 복습 (자료형, 함수, 클래스, 파일 입출력, 예외처리)

데이터 분석 기본기 (Pandas, Numpy, EDA, 시각화)

머신러닝 파이프라인 (사이킷런 중심, metric 계산 습관화)

NLP/CV 실습 (수업 과제 + 개인 프로젝트로 확장)

깃허브·노션 기록 습관화 (디지털 프린트 남기기)

4. 실행 일정 (12주 학기 기준)
주차별 흐름

1–3주차: 파이썬 기초 복습 집중 + Pandas 가볍게 병행

4–6주차: Pandas 심화 + EDA 루틴 정착 + 알고리즘 Easy 문제 시작

7–9주차: ML 파이프라인 학습 + 프로젝트 1개(분류/회귀) 진행

10–12주차: NLP 또는 CV 프로젝트 시작 + 프로젝트 2·3 준비

주간 루틴

평일(1~1.5시간)

알고리즘 1문제 (20분)

파이썬/판다스 복습 or ML 함수화 실습 (30~40분)

수업 내용 함수/파이프라인으로 재작성 (30분)

주말(2~3시간)

미니 프로젝트 실행 (수업 + 개인 주제)

깃허브 커밋, 노션에 정리

----------------------------------------------------------------------------------------------------------

5. 깃허브·노션 운영법

깃허브 = 코드 창고

레포지토리: study-python, study-ml, project-nox, project-nlp …

README.md: 프로젝트 개요, 실행법, 결과 요약 포함

노트북(.ipynb) + 소스(.py) + 결과(/results) 정리

노션 = 설명 문서

카테고리: 📘 Study / 🚀 Projects / 🗂 Portfolio

Study: 파이썬 문법, 알고리즘 풀이 노트, Pandas 정리

Projects: 개요 → 데이터 설명 → 과정 → 결과 → 개선 아이디어

Portfolio: 대표 프로젝트 3~5개 요약, 깃허브 링크 첨부

----------------------------------------------------------------------------------------------------
